# ThreemaWebQt

## Thin web client for threema web using Qt

# Requirements
Qt-5 >= 5.9

QtWebEngine >= 1.7

# Build
Clone the git:
`git clone https://github.com/ethanrakoff/threemawebqt.git`

Use qmake to generate a makefile:
`qmake`

Use make to build:
`make`

Install with make:
`make install`

# Usage
Launch using your desktop environment, or run `threemawebqt` in a terminal.
